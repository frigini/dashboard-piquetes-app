# Dashboard Piquetes App

Este é um projeto React instanciado utilizando o Vite. O aplicativo consiste em um Dashboard completo para acompanhamento de montagens e projetos de piquetes, de uso voltado à exibição e análise de dados.

## Requisitos

- **Node.js**: Certifique-se de ter o Node.js (versão 16 ou superior) instalado em sua máquina. Você pode baixá-lo em [nodejs.org](https://nodejs.org/).

## Instalação

1. Clone este repositório para o seu computador ou, caso já possua os arquivos, abra a pasta `dashboard-app` em seu terminal.
2. Certifique-se de estar na raiz do diretório do projeto onde se encontra o arquivo `package.json`.
3. Execute o comando abaixo para instalar todas as dependências do projeto:

```bash
npm install
```

## Como Executar

Para rodar a aplicação em um servidor local e visualizar as alterações em tempo real, basta certificar-se de ter instalado os pacotes acima e utilizar o seguinte comando:

```bash
npm run dev
```

Após executar este comando, o terminal mostrará um link (como `http://localhost:5173/` ou similar). Segure a tecla `Ctrl` (ou `Cmd` no Mac) e clique no link para abrir diretamente no seu navegador, ou copie a URL e cole na barra de pesquisa de qualquer navegador web.

## Build para Produção

Quando for distribuir o seu projeto final, você pode rodar:

```bash
npm run build
```

O código estático e pronto para a produção será gerado dentro do diretório `dist/`.
